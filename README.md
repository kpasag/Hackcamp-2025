# Welcome to ClashRoom!
![Clash Room Logo](images/ClashRoomFullLogoW.png)

## How to Play
Clash Room is a multiplayer game you can play with your peers during lecture where the objective of the game is to listen up and try and catch the common phrases or things that may happen during the session, After you complete a row or column, the game resets

## Tech Stack
For ClashRoom we used 
- **HTML**
- **Tailwind CSS**
- **JavaScript**
- **Firebase**
